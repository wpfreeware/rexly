== Description ==

Rexly is a multipurpose onepage business theme built using the most modern web technologies. It's suitable for freelancers, portfolios & any other regular businesses.

== Installation ==

Please read theme documentation: https://www.wpfreeware.com/docs/rexly-documentation/

== Frequently Asked Questions ==

= Does this theme support any plugins? =

rexly includes support for the jetpack, Contact form 7, Kirki, CMB2, Yoast SEO, WooCommerce plugins.
